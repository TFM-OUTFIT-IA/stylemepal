Para iniciar la aplicación de Angular se requiere:
1. Node.js (npm)
2. Instalar Angular (`npm install -g @angular/cli`) 
3. Entrar a la carpeta front/frontend mediante terminal y ejecutar `npm install` la primera vez
4. Ejecutar `ng serve` cada vez que se quiera lanzar

Para lanzar la aplicación de Python:
1. Crear entorno virtual de Python `python -m venv venv`
2. Ejecutar el entorno `venv\Scripts\activate`
3. Instalar los requerimientos de requirement.txt `pip install -r requirements.txt`
4. Lanzar la aplicación mediante uvicorn `uvicorn main:app --reload`
